# blockchan

A blockchain explorer built with React and Redux.

## Requirements

Node >= v7.7.2

## Deploy

``` shell
git clone https://github.com/miuchan/blockchan.git
npm install
sudo node publicServer   // cause of listening any port below 1024 requires root access.
```
